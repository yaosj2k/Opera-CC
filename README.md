Opera AG  
=========================

###Suggest 11.64 version
这个版本主要是一些py,bash脚本，而且比较适合宽屏用户  
然后解压到任何一个路径(这是绿色版)，我推荐 $HOME/opt 里面  
  
###安装方法如下：  
+ 下载zip并解压,run  
    > ./setup-11.64.sh
+ 全部OK的话，说明命令都没有问题，可以直接双击opera运行
