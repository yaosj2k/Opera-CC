Opera CC  
=========================

###Suggest 11.64 version
解压到任何一个路径(这是绿色版)  
  
###安装方法如下：  
1. 下载zip并解压,run  
   > ./setup-11.64.sh
2. 全部OK的话，说明命令都没有问题，可以直接双击opera运行
3. 然后用opera打开button.html,按钮都在这里面了,点击就可以添加.
